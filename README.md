## Projeto To-Do List

Durante este desafio, utilizamos os conceitos básicos do React desenvolvendo do zero uma simples lista de tarefas com as opções de criar e apagar um item da lista.

### 🛠️ Nesse projeto foi utilizado

* Vite
* CSS Modules
* Typescript
* Local Storage

## 🚀 Como executar

**Para que esse projeto funcione corretamente, é preciso estar com o servidor rodando.**

- Instale os pacotes com `npm install ou yarn`.
- Execute `npm run dev ou yarn dev` para iniciar o cliente web.
